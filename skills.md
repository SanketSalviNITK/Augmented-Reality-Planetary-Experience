# 🛠️ Required Skills & Technologies

Each agent possesses a specific skill set required to fulfill their role in the project.

## 🛸 Galactic Command (Orchestration)
- **Workflow Management**: Git, Task tracking, System integration.
- **Problem Solving**: Debugging cross-agent dependencies.
- **Quality Assurance**: Code review and performance auditing.

## 🪐 Cosmos Sage (Content)
- **Scientific Research**: NASA/ESA API knowledge, planetary science.
- **Copywriting**: Educational tone-of-voice, micro-copy.
- **Localization**: (Optional) Multi-language support.

## 🎨 Galaxy Designer (Design)
- **UI Design**: Figma/Mockup creation, Color theory, Typography (Outfit/Inter).
- **Modern CSS**: Flexbox/Grid, Animations (Keyframes), Glassmorphism.
- **Print Design**: CMYK vs RGB considerations for PDF markers.

## ☄️ Stellar Architect (AR/3D)
- **AR Frameworks**: AR.js, A-Frame, WebXR API.
- **3D Libraries**: Three.js, Model-Viewer.
- **Optimization**: GLB/GLTF compression (Draco/Meshopt).

## 💻 Universal Builder (Development)
- **Frameworks**: React.js, Vite.
- **PDF Libraries**: jsPDF, React-PDF.
- **State Management**: Context API or Zustand.
