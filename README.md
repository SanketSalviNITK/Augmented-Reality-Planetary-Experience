Welcome to the **Augmented Reality Planetary Experience**! This project is a magical, interactive gateway designed to spark wonder and curiosity in young explorers about our magnificent Solar System.

Using the power of **Web-based Augmented Reality (WebAR)**, we bring the cosmos right into the palms of children's hands, turning any room into a personal observatory—no app installation required!

---

## 🌟 Key Features

- **☄️ Interactive 3D Planets**: Realistic, high-fidelity 3D models of every planet, from the scorching surface of Mercury to the icy rings of Saturn.
- **🖐️ Intuitive Gesture Control**: Specifically designed for small hands—rotate, zoom, and explore planetary surfaces with simple touch gestures.
- **📚 Bite-Sized Learning**: Interactive "Science Hotspots" that trigger fun facts and bite-sized trivia narrated for kids.
- **🔊 Immersive Soundscapes**: Cinematic ambient space music and planetary "sounds" captured by NASA.
- **📏 Scale Visualizer**: A unique "Pocket Solar System" mode that lets kids see scale comparisons (e.g., how many Earths fit inside Jupiter).
- **📸 Snapshot Mode**: Capture photos of planets in your room to share with friends and family.

---

## 🎯 Our Mission

Our goal is to democratize space education. By using WebAR, we remove the friction of app stores, making high-quality interactive learning **accessible, engaging, and unforgettable** for the next generation of astronomers and astronauts.

---

## 🛠️ Technology Stack (Planned)

- **Frontend**: [React.js](https://reactjs.org/) / [Vite](https://vitejs.dev/)
- **3D Engine**: [Three.js](https://threejs.org/) or [A-Frame](https://aframe.io/)
- **AR Framework**: [AR.js](https://ar-js-org.github.io/AR.js-Docs/) or [Model-Viewer](https://modelviewer.dev/)
- **Styling**: Vanilla CSS with modern Glassmorphism aesthetics

## 🚀 Getting Started

> [!NOTE]
> This project is currently in the **Pre-Alpha / Conceptual** phase. We are preparing for the first launch!

### Prerequisites
- A modern smartphone (Android with ARCore or iOS with ARKit)
- A web browser (Chrome, Safari, or Edge)

### Installation (Coming Soon)
```bash
# Clone the repository
git clone https://github.com/SanketSalviNITK/Augmented-Reality-Planetary-Experience.git

# Install dependencies
npm install

# Start the cosmic journey
npm run dev
```
